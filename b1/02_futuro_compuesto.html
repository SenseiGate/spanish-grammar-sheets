CEFR: B1

Lección 2: Futuro compuesto

📚 1. Explicación gramatical（文法の説明）

1) ¿Qué es el futuro compuesto?

未来完了は、「未来のある時点までに完了していること」を表す時制です。

El futuro compuesto se forma con haber en futuro + participio pasado. Se usa para hablar de acciones que estarán terminadas antes de un momento específico del futuro.

「〜までに〜しているだろう」という意味で、未来の中の「さらに前」を表します。

PersonaHaber (futuro)ParticipioEjemplo
yohabréterminadoHabré terminado.
túhabráscomidoHabrás comido.
él/ella/ustedhabrásalidoHabrá salido.
nosotros/ashabremosllegadoHabremos llegado.
vosotros/ashabréisvividoHabréis vivido.
ellos/as/ustedeshabránestudiadoHabrán estudiado.

2) Uso principal: acción completada antes de un punto futuro

未来のある時点までに完了していることを表します。

  • Para 2030 habré terminado mi proyecto.
    2030年までにプロジェクトを終えているでしょう。
  • Cuando vuelvas, ya habré cocinado.
    あなたが戻る頃には、私はもう料理を作っているでしょう。

3) Expresar suposiciones sobre el pasado

過去についての推測にも使われます。「〜したのだろう」。

  • Ya habrá llegado a casa.
    もう家に着いているだろう。
  • Habrá olvidado el teléfono.
    携帯を忘れたのだろう。
✏️ 2. Oraciones cortas de ejemplo（短い例文）

• Para mañana habré terminado el informe.
明日までに報告書を終えているでしょう。

• Cuando llegues, ya habré limpiado la casa.
あなたが来る頃には、家を掃除しているでしょう。

• Habré estudiado todo el vocabulario para el examen.
試験までに語彙を全部勉強しているでしょう。

• Él habrá salido hace unos minutos.
彼は数分前に出たのだろう。

• Para esa fecha ya habremos decidido.
その日までには、もう決めているでしょう。

• Habrán llegado al aeropuerto muy temprano.
彼らはとても早く空港に着いているだろう。

• Yo habré aprendido mucho para entonces.
その頃には、私はたくさん学んでいるでしょう。

📝 3. Oraciones largas de ejemplo（長い例文）

• Para el final del año, habremos completado todas las tareas importantes del proyecto.
年末までに、プロジェクトの重要な作業をすべて完了しているでしょう。

• Cuando vuelvas de tu viaje, ya habré organizado todos los documentos necesarios.
あなたが旅行から戻る頃には、必要な書類をすべて整理しているでしょう。

• Para entonces, ellos habrán tomado una decisión sobre el nuevo plan.
その頃には、彼らは新しい計画について決定しているでしょう。

• Cuando empiece la reunión, ya habré leído todos los informes previos.
会議が始まる頃には、すでにすべての事前資料を読んでいるでしょう。

• Para el próximo mes, habremos avanzado mucho más de lo esperado.
来月までには、予想以上に進んでいるでしょう。

🧩 4. Ejercicios prácticos（練習問題）

Ejercicio A – Elige la opción correcta（正しい形を選びましょう）

1. Para mañana, yo ya _______ el trabajo.

  • a) habré terminado
  • b) he terminado
  • c) terminaré
Respuesta: a) habré terminado

2. Cuando llegues, ellos ya _______.

  • a) habrán salido
  • b) salen
  • c) saldrán
Respuesta: a) habrán salido

3. Para esa fecha, nosotros _______ todo.

  • a) habremos hecho
  • b) hacemos
  • c) haremos
Respuesta: a) habremos hecho

4. Él ya _______ la noticia.

  • a) habrá oído
  • b) oye
  • c) oirá
Respuesta: a) habrá oído

5. Para entonces, tú _______ suficiente información.

  • a) habrás recibido
  • b) recibes
  • c) recibirás
Respuesta: a) habrás recibido

Ejercicio B – Completa la oración（動詞を未来完了にしましょう）

1. Para el lunes, yo ya __________. (terminar)

habré terminado

2. Cuando llegues, ellos ya __________. (volver)

habrán vuelto

3. Para esa fecha, nosotros __________ el informe. (preparar)

habremos preparado

4. Él ya __________ la verdad. (saber)

habrá sabido

5. Para entonces, tú __________ todo. (aprender)

habrás aprendido
📖 5. Texto de lectura（短い読解文）

Un futuro lleno de planes

Laura está organizando su vida para los próximos años. Tiene muchos proyectos y quiere aprovechar cada oportunidad. Para 2030, habrá terminado su carrera universitaria y habrá trabajado en varios países diferentes. También habrá aprendido otros idiomas y habrá conocido a personas de muchas culturas.

Aunque todavía no sabe exactamente dónde vivirá, está segura de que habrá encontrado un lugar donde se sienta cómoda y feliz. Además, para entonces habrá ahorrado suficiente dinero para viajar por el mundo. Laura cree que, si sigue esforzándose, habrá cumplido la mayoría de sus sueños antes de cumplir los treinta.

計画でいっぱいの未来

ローラはこれから数年間の人生を計画しています。彼女にはたくさんのプロジェクトがあり、あらゆるチャンスを活かしたいと思っています。2030年までに大学を卒業し、いくつかの国で働いているでしょう。また、他の言語も習得し、多くの文化の人々と出会っているはずです。

どこに住むことになるかはまだ分かりませんが、快適で幸せに感じられる場所を見つけているでしょう。さらに、その頃には世界を旅するための十分なお金を貯めているはずです。努力を続ければ、30歳になる前にほとんどの夢を叶えているだろうとローラは信じています。

❓ 6. Preguntas de comprensión（読解問題）

1. ¿Qué habrá terminado Laura para 2030?
2030年までにローラは何を終えているでしょうか。

Habrá terminado su carrera universitaria.

2. ¿Qué habrá aprendido?
彼女は何を習得しているでしょうか。

Habrá aprendido otros idiomas.

3. ¿Qué habrá hecho en varios países?
いくつかの国で何をしているでしょうか。

Habrá trabajado en varios países.

4. ¿Qué habrá ahorrado para entonces?
その頃までに何を貯めているでしょうか。

Habrá ahorrado suficiente dinero para viajar.

5. ¿Qué cree Laura sobre sus sueños?
ローラは自分の夢についてどう思っていますか。

Cree que habrá cumplido la mayoría antes de los treinta.
🎯 Resumen（まとめ）
  • El futuro compuesto se forma con haber en futuro + participio.
    未来完了は「haber（未来形）＋過去分詞」で作ります。
  • Se usa para acciones completadas antes de un momento futuro.
    未来のある時点までに完了していることを表します。
  • También expresa suposiciones sobre el pasado.
    過去についての推測にも使われます。
  • Es común con expresiones como para…, ya….
    「para〜」「ya〜」などとよく一緒に使われます。
