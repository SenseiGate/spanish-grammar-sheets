Lección 49: Imperativo afirmativo

📚 1. Explicación gramatical（文法の説明）

El imperativo afirmativo se usa para dar órdenes, instrucciones o invitaciones.

命令形（肯定形）は、指示・命令・お願い・勧誘などに使います。

1) Formación básica

PronombreForma
túhabla / come / vive
él / ella / (usted)hable / coma / viva
nosotros / nosotrashablemos / comamos / vivamos
vosotros / vosotrashablad / comed / vivid
ellos / ellas / (ustedes)hablen / coman / vivan

男性形・女性形・usted・ustedes を必ず明示します。

2) Usos principales

  • Dar instrucciones → Abre el libro.
  • Dar órdenes → Ven aquí.
  • Invitar → Pasa, por favor.

✏️ 2. Oraciones cortas de ejemplo（短い例文）

  • Habla más despacio.
    もっとゆっくり話してください。
  • Come verduras todos los días.
    毎日野菜を食べてください。
  • Él / Ella / (Usted) hable con el profesor.
    彼／彼女／あなた（丁寧）は先生と話してください。
  • Nosotros / Nosotras hablemos mañana.
    私たちは明日話しましょう。
  • Vosotros / Vosotras vivid en paz.
    あなたたちは平和に暮らしてください。
  • Ellos / Ellas / (Ustedes) coman aquí.
    彼ら／彼女たち／あなたがた（丁寧）はここで食べてください。
  • Escucha bien las instrucciones.
    指示をよく聞いてください。

📝 3. Oraciones largas de ejemplo（長い例文）

  • Por favor, habla más despacio porque no entiendo muy bien.
    よく分からないので、もっとゆっくり話してください。
  • Hable con mi compañero si necesita ayuda con la tarea.
    宿題で助けが必要なら、私のクラスメートと話してください。
  • Nosotros / Nosotras comamos juntos mañana en la cafetería.
    明日カフェテリアで一緒に食べましょう。
  • Ellos / Ellas / (Ustedes) hablen con el director antes de entrar.
    入る前に校長と話してください。
  • Ven a mi casa esta tarde y estudiemos juntos.
    今日の午後、私の家に来て、一緒に勉強しましょう。

📖 5. Texto de lectura（短い読解文）

Instrucciones en mi casa

En mi casa siempre damos instrucciones simples para organizarnos. Por ejemplo, mi madre dice: “Limpia tu habitación” o “Come antes de salir”. Mi padre dice: “Habla conmigo si tienes un problema”. Nosotros también usamos el imperativo para invitarnos: “Cenemos juntos esta noche” o “Ven a ver esta película”. Es una forma fácil y directa de comunicarnos.

家での指示

私の家では、整理するためにいつも簡単な指示を出します。例えば、母は「部屋を片付けて」や 「出かける前に食べて」と言います。父は「問題があったら私と話して」と言います。 私たちも招待するときに命令形を使います。「今夜一緒に夕食を食べよう」や 「この映画を見に来て」などです。簡単で直接的なコミュニケーションの方法です。

❓ 6. Preguntas de comprensión（読解問題）

1. ¿Qué dice la madre?
母は何と言っていますか。

"Limpia tu habitación" o "Come antes de salir".

2. ¿Qué dice el padre?
父は何と言っていますか。

"Habla conmigo si tienes un problema".

3. ¿Para qué usan el imperativo?
命令形は何のために使いますか。

Para dar instrucciones e invitaciones.

4. ¿Qué invitación aparece en el texto?
テキストにはどんな招待の表現がありますか。

"Cenemos juntos esta noche".

5. ¿Por qué usan el imperativo?
なぜ命令形を使いますか。

Porque es una forma fácil y directa de comunicarse.

🎯 Resumen（まとめ）

  • Imperativo afirmativo = órdenes, instrucciones, invitaciones.
    肯定の命令形 = 命令・指示・招待。
  • Formas clave: habla, come, vive (tú) / hable, coma, viva (usted) / hablen, coman, vivan (ustedes).
    主な活用形：tú・usted・ustedes。
  • El verbo en imperativo nunca lleva sujeto explícito.
    命令形では主語を書かない。
  • Útil para dar instrucciones diarias y pedir favores.
    日常の指示やお願いに役立つ。
